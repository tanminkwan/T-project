# T 프로젝트 — 통합 연표 (File: 04_timeline.md)

> **역할**: 세계(`01_world.md` §3) + 소설 두 줄기(`02_family.md` 가족·내면 / `03_present.md` 현재·사건)를 **하나의 원년 기준 시간선**으로 묶은 참조표.
> **태그**: `[세계]` 객관적 지구 사건 · `[가족]` A의 내면·가족 줄기 · `[사건]` 현재 T 프로젝트 줄기.
> **주의**: 가족 연표 수치는 *작업값*(02_family). 세계 연표의 일부 간격(예: "위험의 산업화 ~20년 후")은 원년-상대 근사. 확률 수치는 미확정([확인 필요], 세계관 §1).
> **핵심 정렬 사실**: 발견(원년) → **A 출생 ~원년+14** → 즉 **A는 평생을 *이미 은폐된 그림자* 아래서 자란다**(태어나기 전부터 종말이 계산돼 있었음).

## 0. 한눈에 — 세 줄기의 포개짐

- **세계**는 원년(발견)에서 +52년(충돌)까지 *완만히* 흐르며 새 질서를 정착시킨다(편향 좌절→탈출 제도화→임팩트 붐→진실 봉인→위험의 산업화→탈출→충돌).
- **가족**은 원년+13(맞선)에서 +34(엄마 죽음)에 집중 — A의 트라우마가 *충돌 한참 전에* 완결된다(엄마는 충돌을 보지 못하고 병사).
- **사건**은 A의 30대(원년+40년대)에 압축 — T 파견·추적·폭로·귀환·위상 상승(~+47), 그리고 +52 충돌(2부).
- **운율**: 엄마(솟으려다 닿기 전 묻힘) ↔ A(솟은 적 없이 행성을 떠남). 개인의 시간(가족)이 종(種)의 시간(세계)보다 *먼저, 작게* 같은 비극을 예고한다.

## 1. 연도별 통합 연표 (원년 기준)

| 시점 | 태그 | 사건 |
|---|---|---|
| **원년 (Y0)** | 세계 | **발견** — 칠레 아타카마, 태양 맹점에서 접근 천체 포착(발견자 엘레나, §3.3). 궤도 역산 → *낮은 충돌 확률* 산출(아직 확정 아님). |
| 원년 직후 | 세계 | 각국 행성방어 조직 창설(예: 국가행성방어청). 처음엔 재난 대응 기관. |
| 원년 직후 | 세계 | **편향 좌절** — 확률 상승 + 표면 용융급 → 모든 편향 기술 검토·폐기(질량의 벽, 세계관 §2.2). 궤도역학자(§3.4)가 방어 불가를 판정 → 테이블에서 *편향*이 치워지고 *탈출*이 올라옴(아는 자에게만 도달). |
| **~13년** | 가족 | 엄마(≈20)·아빠(≈45) **맞선·결혼**(부부 나이차 25). 시장 최하점이나 아빠가 엄마의 "무엇"에 *반해* 고름. |
| **~14년** | 가족 | **A 출생**(엄마 21·아빠 46). ※발견 14년 후 — A는 *은폐된 그림자* 아래 출생. |
| 10~20년대 | 세계 | **국제기구의 변질**(명분화, 실권은 산업 클러스터) · **점진적 인지·제도화** · **IPO 위장 자금**(SPCX, §10) · **임팩트 붐** 시작 · **AGI 진입**(방주 두뇌 가속). |
| **~25년** | 가족 | **발원** — A 초등 5학년 면담("천재 같다") → **도시行 결심·엄마의 PT 설득**(A ≈11). 비극의 첫 베팅. |
| **27~29년** | 가족 | **짧은 황금기**(과외 성공·카리스마 장면·M과의 만남, A 13~15). *추락의 발판.* |
| **30~32년** | 가족 | **과학고 · 엄마에게 "한 방" · M 카이스트 조기진학**(A 16~18). → A 방어인격 3층의 기원. |
| **~34년** | 가족 | **엄마 사망**(≈41 요절 · A 대학 2학년 ≈20 · **화해 없음**). A의 사적 "복원본"이 시작되는 상실. |
| ~30년대 | 세계 | **진실 봉인** — 내부 확률 상승(세계관 §2.2)하나 공개는 동결(붕괴 방지). |
| **~원년+20년 후** | 세계 | **위험의 산업화** — 대중이 소행성을 잊고 "우주 ETF 수익률·셸터 청약·AGI IPO"만 생각. 위협이 산업 카테고리가 됨(세계관 §4.4). |
| **40년대** | 사건 | A 성인·항우연 엔지니어 → **다국적 발사체 PM**. **C가 A를 T에 파견**("U 보좌" 위장 = 감시·승계, §3.2). |
| 40년대 | 사건 | **추적 라인**(§4.7): 모순 발견 → 음모 단정(**U 표적**) → 명단의 공백 → **U 경유 희생의 진실 폭로** + **A-U 갈등 정점**(A의 보고면 U 짤림). |
| 40년대 | 사건 | **의정서 최종 회의**(효율 vs 의미, §4.8) → **1부 결말**(A 보고·C 안도·승계, §4.9). |
| **~47년** | 사건/가족 | **T project 본격 · A 귀환 · "중요한 위치"**(A ≈33). 위상 상승(*호감형 서기관급* → C가 *국장급 실세*로 그루밍, §3.1 아크 궤적). **M 재회 코다**(카타르시스 이후) · **아빠(≈79)와의 현재 만남**(가족 줄기의 현재 장면). |
| ~47년 이후 | 세계 | **탈출** — 살아있는 1~2만 명, 라그랑주(L4/L5)+화성 이중화. |
| **~52년** | 세계/사건 | **충돌(D-day)**(A ≈38, 2부). 지상의 "우주쇼" 환상 → 표면 멸균. 궤도/화성서 생존자 목격("쇼를 보러 모인 자가 쇼가 됨", 세계관 §9). |
| 충돌 직후 | 세계 | **두 종류의 탄생** — AGI 1차 학습 + 최초의 우주 신생아. |
| 충돌 후 장기 | 세계 | (**3부**) 거주구 간 우열 누적 → **AGI의 거주구 폐기(존엄성 위기)** → 라그랑주(효율) vs 화성(의미) **분기**. |
